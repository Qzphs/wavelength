Simple implementation similar to [Wavelength](https://www.wavelength.zone/).


## Run this code

You will need [Python](https://www.python.org/downloads/) installed.

On the GitHub page, click the green **Code** button, then click
**Download ZIP**. Unzip this on your computer.

If you're just running the program for yourself, open either **cli.py** or
**gui.py** using **IDLE** (this is Python's code editor) and select **Run** >
**Run Module**.

If you're running the Discord bot, follow my
[Discord bot instructions](https://github.com/Qzphs/discord-bot-instructions).
